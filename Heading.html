Heading Tags

In HTML, heading tags are used to define headings or titles within a document of webpage.
They represent the structure of the content by indicating the hierarchy and
importance of different sections.

Types -- There are six types of heading tags that are supported by HTML5 ,
1. h1 tag , 2. h2 tag 3. h3 tag , 4. h4 tag , 5. h5 tag , 6. h6 tag

h1 tag -- has the highest significance priority and it is a main heading then h2 AND so on ..
h6 tag -- has the least significance priority .

However , Each heading tag has their own default size and we can change with
the help of style attribute using css Font-sizeproperty .
For example -- Search engine uses the heading tag to index the structure and content of web pages.

paragraph Tag

Paragraph tag -- is represented in html p tag by which is used to define the paragraph of text within html document .
paragraphs are block level elements , which means typically start on new line and
stretch to full width available to them .
The text inside paragrapgh tags rendered as a single paragraph by the web browser .
Additionally, paragraph tags can contain other inline elements like links, images, and spans,
allowing for rich and flexible formatting of text content within a web page.

HTML Horizontal RUles

The hr tag is a self closing tag which is used to create horizontal rules and
aslo known as break or divider .
It is used to visually separate the content sections or to indicate changes in topic within the html document.
It renders the horizontal line between the diffrent sectons of content , And
It doesn't requires any other attribute , It is a block level element by default ,
So it will not create new block , the content before and after it , will still be in same block .
