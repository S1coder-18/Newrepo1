URLs (Uniform Resource Locators) are addresses used on the web to identify resources 
such as web pages, images, files, and services.       
1.Absolute url - An absolute url provides the complete address of the web page ,
including the protocol (http/https) and domain name and path to resource as well .

Usage - This url is used when we wan to navigate to other website.
For ex -- Google

Relative url -- Relative url provides the path to resource about current documents path ,
without main domain & protocol . 

Usage -- When we want to move from one page to other page in same document .
For ex -- Heading

Anchor url -- Anchor url when we want to move in diffrent sections in same page . 

Usage -- This is useful when there is lot of content in web page and we want 
to move in specfic and important section. 

For ex -- section1
