What is the use of Doctype -- Doctype is not a Tag , it's an instruction to the web browser about what version of HTML the page is written in. It stands for "Document Type Declaration". Its primary purpose is to tell the web browser which version of HTML the page is using, so that the browser can render the page correctly.
